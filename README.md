# (python) modules for MCM

适用于[<u>Python 3.7.2</u>](https://www.python.org/downloads/release/python-372/)
(凸优化库cvxpy仅支持到3.7.2)

```
.
├───README.md
├───comp_eval/
│   └───pre_proc.py ----------- 数据预处理
├───fi/
│   └───fitting.py ------------ 拟合
├───io/
│   └───writer.py ------------- 写文件
└───multi_analysis/
    └───pca.py ---------------- 主成分分析
```
